## ⚙️ Как это работает?

1. 📲 Получаешь пуш с заданием в рандомный момент
2. 💪 Выполняешь челлендж (видео, текст, фото и т.д.)
3. 💸 Платишь небольшой взнос за участие
4. 🧠 Алгоритм выбирает победителей
5. 🏆 Призовой фонд делится между лучшими
